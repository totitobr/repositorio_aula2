# Repositório Aula 2

Bem-vindo ao repositório da **Aula 2**!

## Descrição

Este repositório foi criado para organizar os materiais, exercícios e exemplos utilizados na Aula 2.

## Estrutura do Repositório

```
repositorio_aula2/
├── README.md       # Este arquivo de apresentação
└── exercicios/     # Pasta com os exercícios da aula
```

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/totitobr/repositorio_aula2.git
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd repositorio_aula2
   ```

3. Explore os arquivos e exercícios disponíveis.

## Contribuição

Para contribuir com este repositório:

1. Faça um **fork** do projeto.
2. Crie uma branch com a sua modificação: `git checkout -b minha-modificacao`
3. Faça o **commit** das suas alterações: `git commit -m 'Adiciona nova funcionalidade'`
4. Faça o **push** para a branch: `git push origin minha-modificacao`
5. Abra um **Pull Request**.
